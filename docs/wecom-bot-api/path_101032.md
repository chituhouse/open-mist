# 企业微信智能机器人 — 模板卡片类型

> 原始文档：https://developer.work.weixin.qq.com/document/path/101032
> 最后更新：2025年

---

## 一、概述

模板卡片是智能机器人回复的一种富文本消息形式，支持多种卡片类型，可以承载按钮、图文、投票等交互元素。

**版本兼容性：**
| 卡片类型 | 最低客户端版本 |
|---------|-------------|
| 文本通知型、图文展示型、按钮交互型 | 企业微信 3.1.6+ |
| 投票选择型、多项选择型 | 企业微信 3.1.12+ |

**注意：**
- 微工作台（原企业号）**不支持**展示模板卡片消息
- 仅按钮交互型、投票选择型、多项选择型以及填写了 `action_menu` 字段的文本通知型/图文展示型，支持回调更新或通过接口更新卡片

---

## 二、卡片类型列表

| 卡片类型 | `card_type` 值 | 支持交互 |
|---------|--------------|---------|
| 文本通知型 | `text_notice` | 部分（需配置 `action_menu`） |
| 图文展示型 | `news_notice` | 部分（需配置 `action_menu`） |
| 按钮交互型 | `button_interaction` | 是 |
| 投票选择型 | `vote_interaction` | 是 |
| 多项选择型 | `multiple_interaction` | 是 |

---

## 三、通用结构体说明

以下结构体在多种卡片类型中复用：

### Source（卡片来源）
```json
{
  "icon_url": "https://xxx.cdn.com/2021/xxx.png",
  "desc": "企业名称",
  "desc_color": 1
}
```

| 字段 | 说明 |
|------|------|
| `icon_url` | 来源图标 URL |
| `desc` | 来源描述文字 |
| `desc_color` | 来源文字颜色：`0`=默认黑，`1`=灰，`2`=绿，`3`=橙，`4`=红 |

### ActionMenu（右上角菜单）
```json
{
  "desc": "更多操作",
  "action_list": [
    { "text": "选项一", "key": "action_key_1" },
    { "text": "选项二", "key": "action_key_2" }
  ]
}
```

### MainTitle（主标题）
```json
{
  "title": "主标题内容",
  "desc": "主标题副标题内容"
}
```

### QuoteArea（引用文本区域）
```json
{
  "type": 1,
  "url": "https://www.example.com",
  "title": "引用文本标题",
  "quote_text": "引用文本内容"
}
```

| `type` 值 | 说明 |
|----------|------|
| `1` | 文字引用（点击可跳转） |
| `2` | 文档引用 |

### HorizontalContent（左图右文横向内容）
```json
{
  "keyname": "字段名",
  "value": "字段值",
  "type": 0,
  "url": "https://www.example.com"
}
```

| `type` 值 | 说明 |
|----------|------|
| `0` | 链接（默认） |
| `1` | 链接（用于展示链接） |
| `2` | 文件附件下载 |
| `3` | 员工信息（点击跳转员工详情） |

### CardAction（点击卡片跳转）
```json
{
  "type": 1,
  "url": "https://www.example.com",
  "appid": "wx2b1c6dd1a3f9ab3b",
  "pagepath": "pages/home/index"
}
```

| `type` 值 | 说明 |
|----------|------|
| `1` | 跳转 URL |
| `2` | 跳转小程序 |

### Button（按钮）
```json
{
  "text": "按钮文字",
  "key": "button_key",
  "type": 1,
  "style": 1
}
```

| `type` 值 | 说明 |
|----------|------|
| `1` | 回调按钮（触发回调事件）|

| `style` 值 | 说明 |
|-----------|------|
| `1` | 默认样式 |
| `2` | 边框样式 |
| `3` | 强调样式（蓝色） |

## 四、文本通知型（`text_notice`）

适用于纯文本通知场景。

```json
{
  "msgtype": "template_card",
  "template_card": {
    "card_type": "text_notice",
    "source": {
      "icon_url": "https://xxx.cdn.com/2021/xxx.png",
      "desc": "企业名称",
      "desc_color": 0
    },
    "action_menu": {
      "desc": "更多操作",
      "action_list": [
        { "text": "举报", "key": "action_report" }
      ]
    },
    "main_title": {
      "title": "欢迎使用企业微信",
      "desc": "您的好友正在使用企业微信"
    },
    "emphasis_content": {
      "title": "100",
      "desc": "总计"
    },
    "quote_area": {
      "type": 1,
      "url": "https://work.weixin.qq.com",
      "title": "官方说明",
      "quote_text": "企业微信是一款企业通讯与办公工具。"
    },
    "sub_title_text": "下划线标题",
    "horizontal_content_list": [
      { "keyname": "邀请人", "value": "张三" },
      { "keyname": "企微官网", "value": "点击访问", "type": 1, "url": "https://work.weixin.qq.com" }
    ],
    "jump_list": [
      { "type": 1, "url": "https://work.weixin.qq.com", "title": "企业微信官网" }
    ],
    "card_action": {
      "type": 1,
      "url": "https://work.weixin.qq.com"
    }
  }
}
```

## 五、图文展示型（`news_notice`）

适用于展示图片+文字的信息卡片。

```json
{
  "msgtype": "template_card",
  "template_card": {
    "card_type": "news_notice",
    "source": {
      "icon_url": "https://xxx.cdn.com/2021/xxx.png",
      "desc": "新闻来源"
    },
    "main_title": {
      "title": "今日头条",
      "desc": "点击查看详情"
    },
    "card_image": {
      "url": "https://xxx.cdn.com/image.jpg",
      "aspect_ratio": 2.25
    },
    "image_text_area": {
      "type": 1,
      "url": "https://work.weixin.qq.com",
      "title": "图文区域标题",
      "desc": "图文区域描述",
      "image_url": "https://xxx.cdn.com/small.jpg"
    },
    "vertical_content_list": [
      { "title": "标题一", "desc": "描述一" },
      { "title": "标题二", "desc": "描述二" }
    ],
    "card_action": {
      "type": 1,
      "url": "https://work.weixin.qq.com"
    }
  }
}
```

**`card_image` 字段说明：**

| 字段 | 说明 |
|------|------|
| `url` | 图片 URL |
| `aspect_ratio` | 图片宽高比（宽/高），取值范围 0.5~2.25，默认 1.3 |

## 六、按钮交互型（`button_interaction`）

适用于需要用户点击按钮做出选择的场景，点击后触发回调事件。

```json
{
  "msgtype": "template_card",
  "template_card": {
    "card_type": "button_interaction",
    "source": {
      "desc": "智能助手"
    },
    "main_title": {
      "title": "请选择您的操作",
      "desc": "点击按钮执行对应操作"
    },
    "task_id": "TASK_ID",
    "button_list": [
      {
        "text": "查询天气",
        "key": "query_weather",
        "type": 1,
        "style": 1
      },
      {
        "text": "查询股票",
        "key": "query_stock",
        "type": 1,
        "style": 2
      }
    ],
    "card_action": {
      "type": 1,
      "url": "https://work.weixin.qq.com"
    }
  }
}
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| `task_id` | 卡片任务 ID，用于关联回调事件 |
| `button_list` | 按钮列表（数组） |
| `button_list[].text` | 按钮显示文字 |
| `button_list[].key` | 按钮 key，点击后回调中携带 |
| `button_list[].type` | 按钮类型（目前固定为 `1`，回调类型） |
| `button_list[].style` | 按钮样式：`1`=默认，`2`=边框，`3`=强调 |

## 七、投票选择型（`vote_interaction`）

适用于单项选择场景，用户点击选项后触发回调。

```json
{
  "msgtype": "template_card",
  "template_card": {
    "card_type": "vote_interaction",
    "source": {
      "desc": "智能助手"
    },
    "main_title": {
      "title": "您对本次服务的满意度？"
    },
    "task_id": "TASK_ID",
    "checkbox": {
      "question_key": "satisfaction",
      "option_list": [
        { "id": "very_satisfied", "text": "非常满意", "is_checked": false },
        { "id": "satisfied", "text": "满意", "is_checked": false },
        { "id": "unsatisfied", "text": "不满意", "is_checked": false }
      ],
      "mode": 0
    }
  }
}
```

**`checkbox` 字段说明：**

| 字段 | 说明 |
|------|------|
| `question_key` | 问题 key，回调时携带 |
| `option_list` | 选项列表 |
| `option_list[].id` | 选项 ID，回调时携带 |
| `option_list[].text` | 选项显示文字 |
| `option_list[].is_checked` | 是否默认选中 |
| `mode` | `0`=单选，`1`=多选 |

---

## 八、多项选择型（`multiple_interaction`）

适用于表单收集场景，包含多个下拉选择器和提交按钮。

```json
{
  "msgtype": "template_card",
  "template_card": {
    "card_type": "multiple_interaction",
    "source": {
      "desc": "智能助手"
    },
    "main_title": {
      "title": "请填写信息"
    },
    "task_id": "TASK_ID",
    "select_list": [
      {
        "question_key": "city",
        "title": "城市",
        "selected_id": "guangzhou",
        "disable": false,
        "option_list": [
          { "id": "guangzhou", "text": "广州" },
          { "id": "shenzhen", "text": "深圳" },
          { "id": "beijing", "text": "北京" }
        ]
      }
    ],
    "submit_button": {
      "text": "提交",
      "key": "submit_key"
    },
    "replace_text": "已提交，感谢您的参与！"
  }
}
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| `select_list` | 下拉选择器列表 |
| `select_list[].question_key` | 问题 key，提交时回调携带 |
| `select_list[].title` | 选择器标签文字 |
| `select_list[].selected_id` | 默认选中项的 ID |
| `select_list[].disable` | 是否禁用该选择器 |
| `select_list[].option_list` | 选项列表 |
| `submit_button.text` | 提交按钮文字 |
| `submit_button.key` | 提交按钮 key，回调时携带 |
| `replace_text` | 提交后替换显示的文本 |

---

## 九、卡片更新规则

以下卡片类型支持通过回调或接口更新：
- 按钮交互型
- 投票选择型
- 多项选择型
- 填写了 `action_menu` 字段的文本通知型/图文展示型

更新时可以将按钮更新为不可点击状态，并自定义文案（通过 `button.replace_name` 字段）。
